## Project Context

A personal job search tracker that auto-fetches job listings from companies like Amazon, Google, Tesla, McKinsey, EY, AWS, Intel, Applied Materials, Mastercard, Home Depot, and Rivian via the JSearch API and generates formatted Excel spreadsheets.

**Stack:** Python, openpyxl, requests, JSearch RapidAPI  
**Output:** Company-branded `.xlsx` files with job listings, salary, links, and match analysis
